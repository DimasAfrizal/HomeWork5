Nama Aplikasi : Spotify Playlist Maker
Fitur : Membuat Playlist
Cara Menggunakan :
1. go to https://final-project-olive.vercel.app/
2. Click Login
3. Click Get user data (Make sure u click this button)
4. write ur title also the description for ur Playlist
5. Search ur songs or tracks
6. select ur songs
7. Click Submit Button to make ur Playlist on spotify
